Java Basics

Here are some Java basics with code examples to get you started:

1. Hello World


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. Variables and Data Types

Java has strong data types, meaning that each variable needs to be declared with its type.


int age = 30;
String name = "Alice";
double salary = 85000.50;
boolean isEmployed = true;

System.out.println(age + ", " + name + ", " + salary + ", " + isEmployed);

3. Control Flow (if-else)

Use `if`, `else if`, and `else` to control the flow of your Java programs.


int number = 10;

if (number > 0) {
    System.out.println("Positive number");
} else if (number < 0) {
    System.out.println("Negative number");
} else {
    System.out.println("Zero");
}

4. Loops

Java has different types of loops like `for`, `while`, and `do-while` to repeat actions.


// For loop
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// While loop
int count = 0;
while (count < 5) {
    System.out.println(count);
    count++;
}

5. Methods

Methods in Java allow code to be organized into reusable functions.


public class Main {
    public static void main(String[] args) {
        greet("Alice");
        greet("Bob");
    }

    public static void greet(String name) {
        System.out.println("Hello, " + name);
    }
}
