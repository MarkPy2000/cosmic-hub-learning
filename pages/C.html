Welcome to the C Programming Universe!

C is a general-purpose, procedural computer programming language supporting structured programming, lexical variable scope, and recursion, with a static type system.

C Language Fundamentals

Start your journey with these fundamental C concepts:

  • Language Fundamentals - Learn the basic structure of a C program, syntax, and semantics.
  • Example: 
    
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
  • Statements - Control the flow of your C program with statements.
  • Example: 
    
int main() {
    int x = 10;
    if (x > 5) {
        printf("x is greater than 5");
    }
    return 0;
}
  • Expressions - Learn how expressions are evaluated in C.
  • Example: 
    
int result = 5 + 10 * 2; // result is 25
  • Declarations - Understand variable and function declarations.
  • Example: 
    
int x; // variable declaration
int add(int a, int b); // function declaration

C Standard Library

Explore the C standard library functions and utilities:

  • Input/Output - Handle user input and program output with standard I/O functions.
  • Example: 
    
#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num); // takes input from user
    printf("You entered: %d", num); // prints output
    return 0;
}
  • String Handling - Manipulate strings using standard library functions.
  • Example: 
    
#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    strcat(str1, str2); // concatenates str2 to str1
    printf("%s", str1); // prints "HelloWorld"
    return 0;
}
  • Memory Management - Learn about dynamic memory allocation in C.
  • Example: 
    
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    ptr = (int*) malloc(sizeof(int)); // dynamically allocate memory
    *ptr = 10;
    printf("%d", *ptr); // prints 10
    free(ptr); // free allocated memory
    return 0;
}

Advanced C Topics

Dive deeper into more advanced C programming concepts:

  • Pointers - Master memory manipulation with pointers.
  • Example: 
    
#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a; // pointer to the address of a
    printf("%d", *p); // prints 10 (value at address p)
    return 0;
}
  • Structures - Group different data types into a single entity.
  • Example: 
    
#include <stdio.h>

struct Student {
    char name[50];
    int age;
};

int main() {
    struct Student s1;
    strcpy(s1.name, "John");
    s1.age = 20;
    printf("%s is %d years old", s1.name, s1.age);
    return 0;
}
