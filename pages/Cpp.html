Welcome to the C++ Dimension of the C Universe!

C++ is a powerful, high-performance language that extends the C programming language with features like object-oriented, generic, and functional programming.

C++ Fundamentals

Start your C++ journey with these core concepts:

  • Basic Concepts - Learn about data types, operators, and control structures. For example, here's how to use a simple `if-else` statement:
    • 
int x = 10;
if (x > 5) {
    std::cout << "x is greater than 5";
} else {
    std::cout << "x is 5 or less";
}
  • Objects and Types - C++ introduces the concept of objects that encapsulate data and functions. Example:
    • 
class Car {
public:
    string model;
    void start() {
        std::cout << "Car started!";
    }
};
Car myCar;
myCar.model = "Toyota";
myCar.start();
  • Declarations and Definitions - Learn how to declare variables and functions. Example:
    • 
int add(int a, int b); // Declaration
int add(int a, int b) { // Definition
    return a + b;
}
  • Functions - Functions are reusable blocks of code. Here's how to define a function that calculates the square of a number:
    • 
int square(int x) {
    return x * x;
}

Object-Oriented Programming in C++

Explore object-oriented programming features in C++:

  • Classes and Structs - Classes allow you to create complex data structures with functions and variables. Example:
    • 
class Dog {
    public:
    string name;
    void bark() {
        std::cout << name << " says Woof!";
    }
};
Dog myDog;
myDog.name = "Buddy";
myDog.bark();
  • Inheritance - Inheritance allows one class to inherit properties and behavior from another class:
    • 
class Animal {
public:
    void eat() { std::cout << "Eating"; }
};
class Dog : public Animal {
    void bark() { std::cout << "Woof!"; }
};
  • Polymorphism - Polymorphism allows objects to be treated as instances of their base type. Example:
    • 
class Animal {
public:
    virtual void speak() { std::cout << "Some sound"; }
};
class Dog : public Animal {
public:
    void speak() override { std::cout << "Bark"; }
};
Animal* animal = new Dog();
animal->speak();  // Output: Bark
  • Abstract Classes - Abstract classes provide a way to define an interface without implementing the details. Example:
    • 
class Shape {
public:
    virtual void draw() = 0; // Pure virtual function
};
class Circle : public Shape {
    void draw() override { std::cout << "Drawing a circle"; }
};

Advanced C++ Features

Dive into advanced C++ programming concepts:

  • Templates - Templates allow you to write generic functions and classes. Example of a template function:
    • 
template 
T add(T a, T b) {
    return a + b;
}
std::cout << add(5, 10); // Works for integers
std::cout << add(2.5, 3.5); // Works for floats
  • Exception Handling - Handle errors gracefully using try-catch blocks. Example:
    • 
try {
    int x = -1;
    if (x < 0) throw std::runtime_error("Negative number");
} catch (const std::exception& e) {
    std::cout << e.what();
}
  • Lambda Expressions - Lambda expressions provide a concise way to write anonymous functions. Example:
    • 
auto square = [](int x) { return x * x; };
std::cout << square(5);
  • Move Semantics - Move semantics improve performance by transferring resources rather than copying them. Example:
    • 
std::string str = "Hello";
std::string movedStr = std::move(str);
